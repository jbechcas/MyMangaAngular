# MyMangaAngular

**MyMangaAngular** es una aplicación web desarrollada con Angular que permite a los usuarios explorar y leer mangas en línea.
## Página principales

- **Home**
- **Mangas**
- **Chapters**
- **About**
- **Profile**

## Tecnologías utilizadas

- **Frontend**: Angular
- **Backend**: Strapi y FireBase
